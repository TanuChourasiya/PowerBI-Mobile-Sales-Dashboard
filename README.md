# PowerBI-Mobile-Sales-Dashboard
Interactive Power BI dashboard for mobile sales performance analysis.

**Project Overview**:-
This project analyzes mobile sales data using Power BI to create an interactive dashboard that helps track business performance, sales trends, and regional insights.

**Key Features**
-KPI Cards: Total Sales, Revenue, Orders
-Monthly Sales Trend Analysis
-Brand and Category Performance
-Region-wise Sales Insights
-Interactive Filters

**Tools Used**
Power BI,Power Query,DAX,Excel
